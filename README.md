# Watchdog Tactical

A modern, mobile-friendly Next.js e-commerce website for Watchdog Tactical - a veteran-owned tactical gear company specializing in custom Kydex holsters, duty rigs, and gun parts.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Project Structure

```
app/
  ├── components/      # Reusable UI components
  ├── contexts/       # React contexts (Cart, Wishlist)
  ├── about/          # About page
  ├── products/       # Products page
  ├── contact/        # Contact page
  ├── cart/           # Shopping cart page
  ├── wishlist/       # Wishlist page
  ├── login/          # Login/Register page
  ├── search/         # Search page
  ├── account/        # Account page
  ├── layout.tsx      # Root layout
  ├── page.tsx        # Home page
  └── globals.css     # Global styles
```
